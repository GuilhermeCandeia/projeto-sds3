# Descrição

Aplicação Web para controle de vendas utilizando React e Spring boot com banco de dados relacional. 

## Ferramentas utilizadas

- JDK 11
- STS
- Postman
- Postgresql 12 e pgAdmin
- Heroku CLI
- NPM
- VS Code
- Git
